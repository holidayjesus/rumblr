# Rumblr Build Guide

This guide is for people who have never built a Tauri/Rust app before. Follow the steps in order and run all commands from the project root:

```bash
cd /Users/richardgilboy/Downloads/Rumblr-IRC
```

## What You Need

Rumblr is a Tauri 2 desktop app:

- Frontend: plain HTML, CSS, and JavaScript in `src/`
- Backend: Rust/Tauri code in `src-tauri/`
- Build command: run from the repo root, not from `src-tauri/`

Install these first:

1. **Xcode Command Line Tools**

```bash
xcode-select --install
```

2. **Rust**

Install from <https://rustup.rs>, then restart your terminal.

Check it:

```bash
rustc --version
cargo --version
```

3. **Node.js**

Install Node.js 20 or newer from <https://nodejs.org>.

Check it:

```bash
node --version
npm --version
```

## First-Time Setup

From the project root:

```bash
npm install
```

This installs the Tauri CLI listed in `package.json`.

Do not upload `node_modules/` to GitHub. It is generated by `npm install`.

## Run In Dev Mode

Use this while testing changes:

```bash
npm run dev
```

That starts the Tauri development app.

## Build The Mac App

For a production build:

```bash
npm run build
```

When it finishes, look in:

```text
src-tauri/target/release/bundle/
```

Common outputs are:

```text
src-tauri/target/release/bundle/macos/Rumblr.app
src-tauri/target/release/bundle/dmg/Rumblr_*.dmg
```

The `.dmg` is the best file to upload for a normal GitHub release.

## Release Upload Checklist

Upload release artifacts, not build folders.

Good files to upload:

- `Rumblr_*.dmg`
- zipped `Rumblr.app`, if you need an app zip
- source archive from GitHub

Do not upload:

- `node_modules/`
- `src-tauri/target/`
- `.DS_Store`
- `src-tauri/gen/schemas/`

Those are generated and can be rebuilt.

## Clean Rebuild

If the build acts weird, clean generated files and rebuild:

```bash
rm -rf node_modules src-tauri/target
npm install
npm run build
```

## Optional Tests

Run the regression tests:

```bash
npm test
```

Run the longer soak tests:

```bash
npm run test:soak
```

## Common Problems

### `xcrun: error: invalid active developer path`

Install Xcode Command Line Tools:

```bash
xcode-select --install
```

### `cargo: command not found`

Rust is not installed or your terminal was not restarted after installing it. Install Rust from <https://rustup.rs>, then open a new terminal.

### `tauri: command not found`

Run:

```bash
npm install
```

Then use:

```bash
npm run dev
npm run build
```

### App Will Not Open On macOS

Unsigned local builds can be blocked by Gatekeeper. Right-click the app and choose **Open**.

For public releases, sign and notarize the app before distributing it.

### GitHub Says Too Many Files

You are probably trying to upload generated folders. Remove these before uploading source manually:

```bash
rm -rf node_modules src-tauri/target src-tauri/gen/schemas
find . -name .DS_Store -delete
```

After cleanup, the project source should be small enough to upload.
