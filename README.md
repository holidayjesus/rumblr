# Rumblr Workstation

High-fidelity IRC communication workstation built with Rust and Tauri.

## Architecture
- **Frontend**: Vanilla JS / CSS (CRT Aesthethic)
- **Backend**: Rust (IRC Protocol)
- **Runtime**: Tauri

## Development
- `npm run dev`
- `npm run build`

For a beginner-friendly compile and release guide, see [BUILD.md](BUILD.md).
